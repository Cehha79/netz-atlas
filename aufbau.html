🌍 Wie das Internet auf der Welt aufgebaut ist

Stell es dir wie ein Straßennetz vor: dein Haus, deine Straße, die Autobahn, und die riesigen Kabel über die ganze Erde. Wir gehen Schicht für Schicht von innen nach außen.

Teil 1 — Von deinem Gerät bis zur Welt (die Schichten)

1 📱

Dein Gerät

Handy, Laptop, Fernseher. Jedes bekommt eine Nummer-Adresse, damit man es findet.

Device diss-wais = Gerät IP Address ai-pi a-dress = Geräte-Adresse
↓
2 📶

Dein Heimnetz (Router)

Der Router zu Hause ist die Tür deiner Wohnung ins Internet. WLAN/Kabel verbinden alle deine Geräte mit ihm.

Router ruu-ter = Verteiler / Tür nach draußen Network nett-wörk = Netz
↓
3 🏢

Dein Anbieter (ISP)

Telekom, Vodafone, o2 … Sie betreiben die Straße vor deinem Haus und bringen dich ans große Netz.

ISP ai-ess-pi = Internet-Anbieter (Internet Service Provider) Provider pro-wai-der = Anbieter
↓
4 🛣️

Das Backbone (die Autobahnen)

Dicke Glasfaser-Leitungen zwischen Städten und Ländern. Hier rast der Verkehr von Millionen Menschen.

Backbone bäck-boun = Rückgrat / Haupt-Leitung Fiber fai-ber = Glasfaser
↓
5 🔀

Knotenpunkte (IXP)

Große Kreuzungen, wo viele Anbieter sich treffen und Daten tauschen. Der bekannteste der Welt steht in Frankfurt (DE-CIX).

Exchange iks-tschehndsch = Austausch / Tausch-Punkt IXP = Internet-Knotenpunkt
↓
6 🌊

Tiefsee-Kabel

Über 500 Kabel liegen am Meeresboden zwischen den Kontinenten. ~95 % des weltweiten Verkehrs laufen durch sie — nicht über Satelliten!

Submarine Cable ssab-ma-rien kei-bl = Untersee-Kabel Continent kon-ti-nent = Kontinent
↓
7 🏬

Rechenzentren (Data Center)

Riesige Hallen voller Server. Hier wohnt die Webseite (Google, YouTube, dein Lieblingsshop). Dein Gerät holt sich von hier die Daten.

Server ssör-wer = Diener-Computer Data Center dei-ta ssen-ter = Rechenzentrum

Teil 2 — Das Bild der ganzen Erde

Kontinente (gelbe Punkte) sind durch Kabel im Meer verbunden:

Jeder Punkt = eine Region voller Rechenzentren & Knotenpunkte. Die Linien = Tiefsee-Kabel.

Teil 3 — Was passiert, wenn du eine Seite öffnest?

📱 Du tippst youtube.com → 📖 DNS findet die Adresse (wie ein Telefonbuch) → 📶 Router → 🏢 Anbieter → 🛣️ Autobahn → 🌊 evtl. Kabel ums Meer → 🏬 Server antwortet → 📱 Bild auf deinem Schirm

Das alles dauert oft weniger als eine Sekunde.

Teil 4 — Wichtige Begriffe auf einen Blick

📖

DNS

Das Telefonbuch des Internets: macht aus dem Namen (google.de) die Nummer-Adresse.

📦

Paket / Packet

Daten werden in kleine Päckchen zerlegt und einzeln verschickt — wie viele kleine Briefe.

🤝

Protokoll (TCP/IP)

Die gemeinsamen "Verkehrsregeln", damit alle Geräte sich verstehen.

🔒

HTTPS

Das Schloss-Symbol: die Verbindung ist verschlüsselt = niemand liest mit.

☁️

Cloud

Nur ein schönes Wort für: "fremde Server in einem Rechenzentrum".

⚡

CDN

Kopien der Webseite nah bei dir — damit es schneller lädt.

🧠 In einem Satz

Das Internet ist kein "Ding in der Luft", sondern ein weltweites Netz aus echten Kabeln, Routern und Server-Hallen — verbunden über Autobahnen aus Glasfaser und dicke Kabel auf dem Meeresboden.
